what is Markup Language

*Markup Language* হলো এমন একটি ভাষা যা টেক্সট (লেখা) কে স্ট্রাকচার (গঠন) এবং ফরম্যাট দেওয়ার জন্য ব্যবহার হয়। এটা কম্পিউটারকে বুঝায় কিভাবে টেক্সট বা কন্টেন্ট ডিসপ্লে বা প্রসেস করতে হবে। সহজ উদাহরণ: আপনি যদি একটি বই লিখেন, তাহলে হেডিং, প্যারাগ্রাফ, লিস্ট ইত্যাদি আলাদা করতে চাইবেন। Markup Language এটা নির্দেশ করে দেয়—কোনটি হেডিং, কোনটি সাধারণ টেক্সট। ### Markup Language vs প্রোগ্রামিং ভাষা: Markup Language* (যেমন:HTML, XML ) শুধু স্ট্রাকচার/ডিজাইন নির্দেশ করে। প্রোগ্রামিং ভাষা* (যেমন:Python, Java ) লজিক/ক্যালকুলেশন করতে পারে। সহজ কথায়:* Markup Language হলো টেক্সটকে অর্গানাইজ এবং সুন্দরভাবে দেখানোর "নিয়ম বই"।
